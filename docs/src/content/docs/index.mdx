## Features

<CardGrid>
  <Card title="Minimal API style" icon="rocket">
    Register jobs with delegates, just like you'd register endpoints. Parameters come from DI, arguments, or the job context.
  </Card>
  <Card title="Distributed" icon="random">
    Run multiple nodes. Surefire makes sure each job runs exactly once — no double execution.
  </Card>
  <Card title="Dashboard" icon="laptop">
    Ships with a real-time dashboard. Live log streaming, progress bars, job management, and node monitoring.
  </Card>
  <Card title="Streaming" icon="seti:pipeline">
    Jobs can return `IAsyncEnumerable&lt;T&gt;` and consume streams from other jobs. Build pipelines where data flows between stages as it's produced.
  </Card>
  <Card title="Batteries included" icon="puzzle">
    Retries with backoff, cron scheduling, timeouts, concurrency limits, lifecycle callbacks, and OpenTelemetry traces out of the box.
  </Card>
</CardGrid>

## Example

export const sample = `using Surefire;
using Surefire.Dashboard;

var builder = WebApplication.CreateBuilder(args);

builder.Services.AddSurefire();

var app = builder.Build();

app.AddJob("Add", (int a, int b) => a + b);

app.AddJob("DataImport", async (JobContext ctx, ILogger<Program> logger, CancellationToken ct) =>
{
    for (var i = 1; i <= 10; i++)
    {
        logger.LogInformation("Step {Step}/10", i);
        await ctx.ReportProgressAsync(i / 10.0);
        await Task.Delay(1000, ct);
    }
}).WithDescription("Imports data and reports progress")
  .WithCron("* * * * *");

app.MapSurefireDashboard();

app.Run();`;
